1. 이진 탐색 : 시작점, 끝점, 그리고 중간점으로 찾으려는 데이터와 중간점 위치에 있는 데이터를 반복적으로 비교하며 원하는 데이터를 찾는 과정

2. 순차 탐색의 시간 복잡도는 O(N)인 반면 이진 탐색의 시간 복잡도는 O(logN)이다. 

3. 이진 탐색의 틀을 항상 외우고 있는 것이 좋다.
-> 크게 2가지 틀이 있다.
